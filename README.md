## RideProgressView
The image runs along the Linear progress bar. With Swift

## Installation
<b>CocoaPods:</b>
<pre>
pod 'RideProgressView'
</pre>
<b>Manual:</b>
<pre>
Copy <i>RideProgressView.swift</i> to your project.
</pre>

## Preview
![](/demo.gif)

## Using

```swift
rideProgressView.setProgress(70, animated: true)
```
